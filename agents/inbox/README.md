# Agent Inbox

PM writes ready-to-run prompt files here.

Almas then gives a specific prompt path to Backend, Frontend, or QA, for example:

```text
Complete the task written in agents/inbox/YYYY-MM-DD_task-name_backend_prompt.md
```

Recommended filenames:

- `YYYY-MM-DD_task-name_backend_prompt.md`
- `YYYY-MM-DD_task-name_frontend_prompt.md`
- `YYYY-MM-DD_task-name_qa_prompt.md`

Do not store secrets, credentials, private scratch notes, or large generated artifacts here.
