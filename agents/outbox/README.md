# Agent Outbox

Backend, Frontend, and QA write response files here after completing or blocking on an assigned prompt.

Almas then tells PM which response file to read.

Recommended filenames:

- `YYYY-MM-DD_task-name_backend_response.md`
- `YYYY-MM-DD_task-name_frontend_response.md`
- `YYYY-MM-DD_task-name_qa_response.md`

Each response should include what changed, checks run, blockers, and anything PM should add to `PROJECT_CONTEXT.md`.

Do not store secrets, credentials, private scratch notes, or large generated artifacts here.
